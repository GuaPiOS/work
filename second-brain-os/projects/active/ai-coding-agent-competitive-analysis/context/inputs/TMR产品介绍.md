# TMR 是什么产品？

**TMR是畜牧养殖领域的全混合日粮（Total Mixed Ration）相关产品，在我司业务体系中包含设备及配套智能解决方案，具体信息如下** ：

1. **基础定义** TMR全混合日粮是一种为反刍动物（如奶牛）配制的均衡饲料，通过专用设备将粗饲料、精饲料、矿物质等原料精准混合而成，能保证动物每一口饲料的营养均衡。对应的TMR搅拌机是实现该日粮配制的核心设备，分为自走式、挂车式（拖拉机牵引）、固定式三类 。
2. **我司TMR相关智能产品**

- **TMR智能饲喂管家（Sveaverken FeedLogger系列）** 是我司推出的TMR饲喂数字化管理系统，能将TMR饲喂流程全面线上化，实现饲料配方设置、任务分配、进度追踪、策略优化的全流程管控，还可提供实时监控、数据记录、异常告警和报表生成功能，具备配方、用量、时间、操作人员的全流程追溯能力 。
  - **产品配置**
    - FeedLogger基础款：适用于自走式/挂车式TMR搅拌机，包含1个控制盒平板、1个LED大屏及对应软件；固定式版本包含2个控制盒平板、1个LED大屏及对应软件 。
    - FeedLogger Pro款：在基础款基础上新增PLC控制精料塔、IoT网关、离线缓存等功能，包含控制硬件、显示屏、电气柜、网关及对应软件，支持自走式/挂车式/固定式TMR搅拌机 。
  - **核心价值** 可减少5-10%的饲料浪费，提升2-5%的产奶量/干物质采食量，每天节省1-2小时的人工时间 。
- **TMR优化套件** 可与推料机器人Nimbo X配合使用，激活后能让Nimbo X扫描追踪剩余饲料并生成实时报告，还可控制Nimbo X调整饲料推送时间，避免与搅拌车作业重叠，优化饲喂策略、减少浪费、提升产奶量 。

1. **相关认证信息** 我司TMR智能饲喂管家（型号TM100，物料号J0300012）曾申请CE/ROHS/Reach/WEEE认证，该产品支持2.4G Wi-Fi、BLE5.0低功耗蓝牙和4G通讯技术 。
