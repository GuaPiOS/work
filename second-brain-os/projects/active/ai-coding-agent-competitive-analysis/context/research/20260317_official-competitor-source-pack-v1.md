# Official Competitor Source Pack v1

## Scope

本笔记只记录第一轮来自官方站点 / 官方资料的竞品事实，优先覆盖产品形态、自动化范围、导航方式、数字化能力和对 `FS200` 的参考价值。

## Competitors

### KUHN AURA

- 官方资料显示其定位为自动 / 自主自走式搅拌投喂设备，覆盖装载、称重、混料、分料、推料和作业报告。
- 资料强调其基于 `GPS + RTK` 进行户外定位，是最接近“自主 TMR 整机”路线的样本。
- 对 `FS200` 的意义：它更像“整机终局形态”的 benchmark，而不是导航套件 retrofit 路线。

### Lely Vector

- 官方资料显示其是完整自动饲喂系统，包含饲料厨房、混料 / 饲喂机器人和料槽高度感知。
- 其核心价值是高频新鲜投喂和整套工作流自动化，而不是单一导航模块。
- 对 `FS200` 的意义：它定义了“完整自动饲喂系统”的价值上限，但和导航套件式改造不是同一层产品。

### GEA DairyFeed F4500

- 官方资料显示其是全电动自动饲喂机器人，强调激光扫描建图、自动导航和较低基础设施改造要求。
- 资料给出其适配大约 `300` 头牛、每小时约 `2.2 m3` 新鲜 TMR 的能力口径。
- 对 `FS200` 的意义：它是“自动机器人 + 导航 + 饲喂闭环”的强 benchmark，但仍偏完整系统。

### Trioliet Triomatic

- 官方资料显示其是成熟的自动饲喂系统，核心卖点是 `24/7` 投喂、较低人工投入和小批次精确混料。
- 官方 brochure 把它表述为能够显著降低每日喂料时间，并支持较小批量起配。
- 对 `FS200` 的意义：它说明市场对“连续投喂 + 降人工”的价值已经成熟，但不直接回答 retrofit 导航套件问题。

### Wasserbauer Shuttle Eco

- 官方资料显示其是电池驱动的自动喂料机器人，强调全天候喂料、约 `2.2 m3` 容量和多圈舍运行。
- 公开资料显示其路线更偏预设路径 / 专用机器人范式，而不是通用导航套件。
- 对 `FS200` 的意义：它是“专用喂料机器人”的对标样本，可用于和通用导航底座路线对照。

### Pellon TMR Robot

- 官方资料显示其是轨道式 / 定轨式 `TMR` 喂料机器人，适合小批次混料与投喂。
- 其路径是室内固定基础设施路线，不是面向存量车辆的自由导航方案。
- 对 `FS200` 的意义：它是不同技术架构的参考样本，说明并非所有自动饲喂都以自由导航为核心。

### Faresin

- 官方资料显示其数字化和自动化能力集中在 `Farmatics`、`Farm Manager Web`、`Ecotronic` 等系统上，覆盖远程监控、配方 / 库存管理、自动执行和智能辅助。
- 这类能力并不等于全无人导航，但对 TMR 主机的数字化、自动化和数据闭环很强。
- 对 `FS200` 的意义：它是“数字化 TMR 主机生态”的高相关参考组，特别适合看软件层和自动化协同。

### SILOKING

- 官方资料显示其 `Feeding Management` 和 `SILOKING Data / Wireless` 覆盖配方、客户管理、卸料程序、循环管理和数据同步。
- `SelfLine 4.0` 官方资料则体现了自走式 TMR 主机与数据系统的结合能力。
- 对 `FS200` 的意义：它是“自走式 TMR + 数字化管理”路线的高相关参考组，但公开资料中没有看到自由导航套件定位。

## First-Pass Structure

1. 完整自动饲喂系统 / 机器人组：
   `Lely Vector`、`GEA DairyFeed F4500`、`Trioliet Triomatic`、`Wasserbauer Shuttle Eco`、`Pellon TMR Robot`
2. 无人自走式 TMR 整机组：
   `KUHN AURA`
3. TMR 主机 + 数字化参考组：
   `Faresin`、`SILOKING`

## What This Means For FS200

- `FS200` 如果定义为 `TMR` 场景下的全无人导航套件，那么它最接近的不是完整自动饲喂系统，而是“自主整机能力”与“数字化 TMR 主机能力”之间的中间层。
- `KUHN AURA` 更接近整机终局能力参考。
- `Faresin` 和 `SILOKING` 更接近数字化和主机协同参考。
- `Lely`、`GEA`、`Trioliet`、`Wasserbauer`、`Pellon` 则定义了完整自动饲喂系统对客户价值的上限。
